# Librarian AI

A comprehensive library assistant system that combines vector-based semantic search with Large Language Models (LLMs) to help students discover books.

## Deployment Note
This Space runs via Docker. Ensure that `library_database.db` is uploaded to the root directory for the application to function correctly.
